The Schulich Hub

The App

The Schulich Hub is a social media app that’s currently in development. It’s meant to be a platform for students to connect and get involved with the community, and will be managed by the Engineering Students’ Society. You can see the project's GitHub here

What I've Learned

I’m a front end developer on the Schulich Hub team. I work mainly with user interfaces and styling the app. Some of the skills I’ve picked up from this role are:

  • UI development using the React framework and Ionic
  • Styling the app using CSS/SCSS
  • Version control with Git/GitHub
  • Collaborating with others on programming and design
  • Attending weekly meetings

This project has really helped me grow my teamwork skills and has given me another real life application of the software development cycle. It’s been amazing to collaborate with my peers on something that will help students for years to come!
