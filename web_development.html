Web Development

Following my experience on the software side of programming, I got involved in web development soon after. I have basic knowledge of HTML, CSS, Javascript. Recently I’ve been developing my skills using the React framework through working on a social media app for the Engineering Students’ Society (ESS) called the Schulich Hub.

Through the Hub, I've also developed my skills working as part of a team in web development. I’ve gotten used to collaborating with other UI designers, backend developers, and full stack developers to build a product.

This has helped me sharpen my communication skills, get used to working with version control tools such as Git and GitHub with groups of people, and taught me about the software development cycle.

Developing this very website was another project of mine. Both the Schulich Hub and this portfolio have made me develop my eye for user interfaces. A challenging aspect of web development is pushing out a product that not only performs well but looks appealing too, and this is what I strive to do!
