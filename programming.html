Programming

Having knowledge of Python and C/C++, I sit in front of the computer more than I'd like to admit! I enjoy coding, both as a hobby and as a learning venture. I’ve taken multiple coding courses and am pursuing a minor in digital engineering as a result.

I love to solve problems through coding, making simple scripts and learning concepts online. This has developed my independence; Google is often my best friend as I search for the answers and learn while bumping into problems trying to debug my code.

Some examples of a small projects I’ve done include:

  • A flight seat map simulator in C++
  • A tic-tac-toe game programmed in Python
  • A grade calculator made using Python

I’m the type of coder who loves his white space and organization. Looking at messy code hurts my eyes, so I take the time to make it neat and well documented!

I find that solving the problems I have drives new ideas which can solve more problems. I’m inspired to take further action to program more, which continues this cycle. This leads to innovative and efficient code over time!
