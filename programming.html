Having knowledge of Python and C/C++, I sit in front of the computer more than I'd like to admit! I enjoy coding, both as a hobby and as a learning venture. I’ve taken multiple coding courses and am pursuing a minor in digital engineering as a result. I like to experiment on my own, making simple scripts and learning concepts online. This has developed my independence; Google is often my best friend as I search for the answers and learn while bumping into problems trying to debug my code.

Some examples of a small projects I’ve done include:

  • A flight seat map simulator in C++
  • A tic-tac-toe game programmed in Python
  • A grade calculator made using Python

What I really love about coding is being able to solve problems with the programs I write. When I write code it’s as if I’m expressing myself through the syntax. I feel like an artist using their pen or an author writing their next award winning book. I find that solving the problems I have drives new ideas which can solve more problems; I’m inspired to take further action, revise what I already have, and program more, which continues the cycle. This leads to innovative and efficient code over time!
